# WeatherSensor
Remote weather sensor project
with 433MHz transmitter and receiver

Material Needed:
Arduino Nano x2
Breadboard - 400 contacts x2
BME280 5V I2C Temeperature Humidity and Baro sensor
433MHz transmitter module
433MHz receiver module


Require Library:
RadioHead
forcedBMX280
BMD31M090
